NoBSjs is an open sourced, boilerplate codebase that aims to reduce the costs of web applications for small businesses and organizations. NoBSjs is built with AngularJS and Angular Material on the frontend and Node, Express, and SQL on the backend.

Speed up development

The NoBSjs codebase comes with all the standard features a web application needs. This allows developers to focus on the unique aspects of a project and deliver better products.

Modules

Every feature in the NoBSjs codebase is modularized. This means modules can be added and removed without affecting the functionality of the application.

Open Sourced

NoBSjs is an open source project. We encourage developers to fork our repo, build new modules, and fix bugs.

Modules

NoBSjs is built using modules. There is a core module that contains features that are common to nearly every web application. Other modules are built on top of the core module, but should remain completely independent from others. There is currently a blog module, but modules for payments, scheduling, inventory, and classes are planned for future releases. If you add modules to the project, please create a pull request!

Core Modules
Core
Authentication

Auth for users is included in the core module. Users can sign up, log in, and reset passwords right out of the box. Auth utilizes JWT tokens for secure session management.

Core
Pages

Allows administrative users to add new pages to the site and edit content.

Core
Tabs

Allows administrative users to modify what links are visible on the navigation bar.

Core
Manage Users

Allow administrative users to manage all users of a site. Edit user information and roles from the admin panel.

Current Modules
Blog

Allows administrators to create blog posts using a WYSIWYG.

Future Modules

These are modules that have been requested but that are not yet built. Let us know if you are interested in tackling a module, we would love help from the community!

Payments

Allow businesses and organizations to accept payments from customers.

Events

Allows businesses and organizations to keep track of a schedule of events and invite customers.

Inventory
