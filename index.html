For businesses developers organizations getting shit done

NoBSjs is an open sourced, boilerplate codebase that helps developers create robust web applications for small businesses and organizations. NoBSjs is built with AngularJS and Angular Material on the frontend and Node, Express, and SQL on the backend.

Speed up development

The NoBSjs codebase comes with all the standard features a web application needs. This allows developers to focus on the unique aspects of a project and deliver better products.

Modulars

Every feature in the NoBSjs codebase is completely modularized. This means modules can be added and removed without affecting any other functionality of the application.

Open Sourced

NoBSjs is an open source project. We encourage developers to fork our repo, build new modules, and fix bugs.
