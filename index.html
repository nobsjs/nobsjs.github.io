NoBSjs is an open source, boilerplate codebase that aims to reduce the costs of web applications for small businesses and organizations. NoBSjs is built with AngularJS and Angular Material on the frontend and Node, Express, and SQL on the backend.

Fast development

All the standard features a web application needs are included so that you can focus on the unique aspects of a project and deliver better products.

Modules

Every feature in the NoBSjs codebase is modularized and modules can be added and removed without affecting the functionality of the application.

Open Source

NoBSjs is an open source project. We encourage developers to fork our repo, build new modules, and fix bugs.

Modules

NoBSjs is built using modules. There is a core module that contains features that are common to nearly every web application. Other modules are built on top of the core module, but should remain completely independent from others. There is currently a blog module, but modules for payments, scheduling, inventory, and classes are planned for future releases. If you add modules to the project, please create a pull request!
